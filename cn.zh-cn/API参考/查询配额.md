# 查询配额<a name="ZH-CN_TOPIC_0020099236"></a>

## 功能介绍<a name="section66578044"></a>

查询用户可以创建的资源配额总数及当前使用量，当前仅有告警规则一种资源类型。

## URI<a name="section62331491"></a>

-   URI格式

    GET /V1.0/\{project\_id\}/quotas
