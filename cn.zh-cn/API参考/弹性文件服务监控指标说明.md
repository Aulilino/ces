# 弹性文件服务监控指标说明<a name="ZH-CN_TOPIC_0077497448"></a>

## 功能说明<a name="section48080847153328"></a>

本节定义了弹性文件服务上报用户请求次数的监控指标的命名空间，监控指标列表和维度定义，用户可以通过云监控提供的API接口来检索用户请求弹性文件服务产生的监控指标和告警信息。

## 命名空间<a name="section20110798153328"></a>

SYS.SFS
