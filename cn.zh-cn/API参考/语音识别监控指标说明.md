# 语音识别监控指标说明<a name="ZH-CN_TOPIC_0137158204"></a>

## 功能说明<a name="section59820001153251"></a>

本节定义了语音识别上报云监控的监控指标的命名空间，监控指标列表和维度定义，用户可以通过云监控提供的API接口来检索语音识别产生的监控指标。

## 命名空间<a name="section172651386227"></a>

SYS.ASR
